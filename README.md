# Personal Portfolio Website

## Project Overview

This project is a personal portfolio website created using HTML and CSS. It introduces me, showcases my programming skills, and provides my contact information through a simple, modern, and responsive design.

## Development Process

1. First, I selected the programming languages and technologies I wanted to showcase. Then, I gathered the required information, planned the website structure, and sketched the layout.

2. After completing the planning phase, I built the website structure using HTML and then styled it using CSS.

3. In the CSS file, I used several features and techniques to improve the appearance and user experience, including:
- Flexbox for layout.
- Linear Gradient background.
- Box Shadow for depth.
- Border Radius for smooth corners.
- Hover Effects on buttons and cards.
- CSS Transitions for smooth animations.
- CSS Transform (Scale and Translate).
- Keyframes Animation for page elements.
- Glassmorphism-inspired cards.
- Responsive spacing and modern color palette.

## Technologies Used

- HTML5
- CSS3
